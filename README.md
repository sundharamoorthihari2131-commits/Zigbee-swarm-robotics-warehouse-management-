# ZigBee-Enabled Swarm Robotics Warehouse Management System

## 📌 Project Overview
This project presents an intelligent warehouse automation system using swarm robotics principles and ZigBee wireless communication. Multiple robots coordinate in real time for obstacle avoidance, hazard detection, material handling and remote monitoring.

## 🚀 Key Features
- ZigBee based inter-robot communication
- Autonomous obstacle avoidance
- Fire hazard detection and alert system
- Accelerometer based motion tracking
- Bluetooth mobile app monitoring
- Decentralized swarm robot coordination

## 🛠 Hardware Components
- Arduino UNO
- ZigBee/XBee Module
- HC-05 Bluetooth Module
- Ultrasonic Sensor
- Fire Sensor
- ADXL345 Accelerometer
- L298N Motor Driver
- DC Motors
- Buzzer

## 💻 Software Used
- Arduino IDE
- Embedded C
- Bluetooth Mobile Interface
